Interactive tool for designing bead patterns.

== Description ==

Bead Pattern Designer is a plugin that allows you to easily design bead craft patterns.

**Key Features:**

* Support for Peyote, Brick, and Square stitches
* Diverse color palette
* Undo/Redo functionality
* Save and load patterns
* Export as image
* Responsive design

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bead-pattern-designer` directory.
2. Activate the plugin through the WordPress admin panel.
3. Add the `[bead-pattern]` shortcode to your page or post.

== Usage ==

**Basic Usage:**
`[bead-pattern]`

**Adjust Size:**
`[bead-pattern width="100%" height="600px"]`

**Custom CSS Class:**
`[bead-pattern class="my-custom-class"]`

== Frequently Asked Questions ==

= Can I add multiple designers on one page? =
Yes, each instance works independently.

= Does it work on mobile? =
Yes, it's responsive and works on all devices.
