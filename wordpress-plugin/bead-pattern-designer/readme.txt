비드 패턴을 디자인할 수 있는 인터랙티브 도구입니다.

== Description ==

Bead Pattern Designer는 비드 공예 패턴을 쉽게 디자인할 수 있는 플러그인입니다.

**주요 기능:**

* Peyote, Brick, Square 스티치 지원
* 다양한 색상 팔레트
* 실행 취소/다시 실행
* 패턴 저장 및 불러오기
* 이미지로 내보내기
* 반응형 디자인

== Installation ==

1. 플러그인 파일을 `/wp-content/plugins/bead-pattern-designer` 디렉토리에 업로드합니다.
2. 워드프레스 관리자 페이지에서 플러그인을 활성화합니다.
3. 페이지나 포스트에 `[bead-pattern]` 숏코드를 추가합니다.

== Usage ==

**기본 사용법:**
`[bead-pattern]`

**크기 조정:**
`[bead-pattern width="100%" height="600px"]`

**커스텀 CSS 클래스:**
`[bead-pattern class="my-custom-class"]`

== Frequently Asked Questions ==

= 여러 개의 디자이너를 한 페이지에 삽입할 수 있나요? =
네, 각 인스턴스는 독립적으로 작동합니다.

= 모바일에서도 작동하나요? =
네, 반응형으로 제작되어 모든 기기에서 사용 가능합니다.
